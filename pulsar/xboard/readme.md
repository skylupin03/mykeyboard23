### Subsequent Flashing

For repeating Flashing you can use BootMagic Lite has been enabled by default and assigned to the "Esc" key:

1. Unplug USB Cable.
2. Hold the "Esc" key.
2. Plug in the USB Cable, the Keyboard should be in ST-Bootloader state.
